# 变更日志

本项目的所有重要变更都会在此文件中记录。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
项目版本遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.2.0] - 2024-XX-XX

### 新增
- 新增 AI 智能助手功能，支持自然语言命令解析
- 新增 OpenAI API 集成支持
- 新增快捷命令 `fbt`（file-batch-tool 的简写）
- 新增示例代码目录，包含多个实用用例
- 新增 API 文档（API.md）
- 新增完整教程文档（TUTORIAL.md）
- 新增贡献指南（CONTRIBUTING.md）
- 新增 CHANGELOG.md 变更日志

### 改进
- 大幅优化文档，增加详细的参数说明和示例
- 优化 setup.py 配置，增加更多分类和元数据
- 优化 README.md，增加使用技巧和社区支持信息
- 为所有模块添加更完善的文档字符串
- 优化错误处理，提供更友好的错误提示

### 修复
- 修复路径字符串格式化问题
- 修复水印透明度参数未使用问题
- 修复 Unicode 编码错误处理

## [1.1.0] - 2024-XX-XX

### 新增
- 新增程序启动自动全屏功能
- 新增进度条显示功能
- 新增文件操作的实时日志输出

### 改进
- 优化 GUI 界面布局
- 改进错误处理机制
- 优化文件操作性能

## [1.0.0] - 2024-XX-XX

### 新增
- 初始版本发布
- 支持批量重命名文件
- 支持图片格式转换（JPG/PNG/WebP/BMP/GIF）
- 支持文件压缩为 ZIP 格式
- 支持按扩展名或日期分类文件
- 支持添加文字/图片水印
- 支持批量修改文件时间
- 支持提取图片 EXIF 信息

- PyQt5 图形界面
- 跨平台支持（Windows/Linux/macOS）

## 版本说明

### 语义化版本
- MAJOR 主版本号：不兼容的 API 修改
- MINOR 次版本号：向下兼容的功能性新增
- PATCH 修订号：向下兼容的问题修正

### 变更类型
- `新增`: 新功能
- `改进`: 对现有功能的改进
- `修复`: Bug 修复
- `移除`: 删除功能
- `废弃`: 即将删除
- `安全`: 安全相关修复
