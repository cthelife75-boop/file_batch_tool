# 文件批量处理工具 📁
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一款基于 PyQt5 开发的跨平台图形化文件批量处理工具，无需命令行操作，纯可视化界面，支持文件重命名、图片格式转换、文件压缩、文件分类、图片加水印等高频办公场景需求，适配 Windows 系统，新手友好。

## ✨ 核心特性
- 🎨 **纯图形化界面**：替代命令行操作，点击选择即可完成所有配置，无需记参数
- 🚀 **多线程处理**：后台执行任务，界面不卡顿，进度条实时显示处理进度
- 📊 **完整日志输出**：所有操作日志可视化，可回溯执行过程和结果
- 💾 **安全处理**：所有处理结果生成新文件，不覆盖原文件，避免数据丢失
- 🔍 **格式全兼容**：支持大小写扩展名，自动处理图片透明通道等兼容性问题

## 📋 功能清单
| 功能模块       | 具体能力                                                                 |
|----------------|--------------------------------------------------------------------------|
| 批量重命名     | 前缀/后缀添加、正则表达式替换、自动跳过已存在文件                         |
| 图片格式转换   | JPG/PNG/WEBP 互转、自动处理 RGBA 透明通道、JPG 自动填充白色背景          |
| 文件压缩       | 批量压缩为 ZIP 包、自定义输出路径、排除指定扩展名文件                     |
| 文件分类       | 按扩展名分类（如 txt/jpg/exe）、按创建日期分类（如 2024-01/2024-02）|
| 图片加水印     | 文字水印（自定义字体/大小/颜色/透明度）、图片水印（自定义尺寸/透明度）|
| 批量修改文件时间 | 修改文件的创建时间 / 修改时间为指定时间 |
| 提取图片 EXIF 信息     | 批量导出图片的拍摄时间、分辨率、设备等信息到 CSV |
| 文件批量复制 / 移动     | 将指定目录文件批量复制 / 移动到目标目录 |


## 📥 快速使用
### 方式1：直接下载可执行文件（推荐，无需编程环境）
1. 进入仓库 [Releases 页面](https://gitee.com/the-life/file_batch_tool/releases)
2. 下载最新版本的 `文件批量处理工具_v1.0.exe`（或对应版本号）
3. 双击 exe 文件即可运行，无需安装 Python 或任何依赖

### 方式2：源码运行（适合开发者/自定义修改）
#### 环境要求
- Python 3.8+（推荐 3.9/3.10）
- Windows/Linux/macOS（Windows 体验最佳）

#### 安装依赖
```bash
# 克隆仓库
git clone https://gitee.com/the-life/file_batch_tool.git
cd file_batch_tool

# 安装核心依赖
pip install -r requirements.txt
# 运行程序
python main.py
```

📖 详细使用教程
#1. 批量重命名<br>
切换至「批量重命名」标签页，点击「选择目录」选择待处理文件所在目录
按需配置：前缀（统一添加文件名开头）、后缀（统一添加文件名末尾）、正则匹配 / 替换（按规则修改文件名）
点击「开始重命名」，等待进度条完成，日志区查看处理结果（自动跳过重名文件）
#2. 图片格式转换<br>
切换至「图片格式转换」标签页，选择图片所在目录
下拉选择目标格式（JPG/PNG/WEBP），点击「开始转换」
转换后的图片自动添加_converted后缀，原图片保留；转换为 JPG 时，透明区域自动填充白色
#3. 文件压缩<br>
切换至「文件压缩」标签页，选择待压缩文件所在目录
可选配置：自定义压缩包保存路径（默认生成「目录名_compressed.zip」）、排除扩展名（逗号分隔，如zip,log,tmp）
点击「开始压缩」，完成后日志区显示压缩包绝对保存路径
#4. 文件分类<br>
切换至「文件分类」标签页，选择待整理文件所在目录
下拉选择分类模式：按扩展名（生成txt/jpg/exe等子文件夹）、按创建日期（生成202X-XX年月子文件夹）
点击「开始分类」，所有文件自动归档至目录下的classified子文件夹
#5. 图片加水印<br>
文字水印<br>
切换至「图片加水印」标签页，默认选择「text（文字水印）」
选择图片目录，填写水印文字，按需配置：字体文件（TTF/OTF 格式，默认系统字体）、文字大小、RGBA 颜色、透明度
点击「开始加水印」，水印默认添加在图片右下角，处理后图片添加watermarked_前缀<br>
图片水印<br>
切换水印类型为「image（图片水印）」，选择图片目录
点击「选择图片」添加水印图片（推荐 PNG 透明格式），配置水印尺寸、透明度
点击「开始加水印」，完成后日志区查看处理结果<br>
# 批量修改文件时间
切换至「批量修改文件时间」标签页，默认选择「both（创建+修改）」时间修改类型
选择文件目录，点击目标时间输入框，按需选择**具体的年/月/日/时/分/秒**，也可直接手动输入时间
按需切换修改类型：仅修改创建时间、仅修改修改时间、两者同时修改
点击「开始修改」，文件时间将统一替换为目标时间，处理后日志区查看处理结果

# 提取图片EXIF信息
切换至「提取图片EXIF信息」标签页
选择图片目录，按需配置CSV保存路径（默认生成`目录名_exif.csv`文件，保存在同目录下）
点击「开始提取」，自动解析图片的拍摄时间、设备型号、分辨率、GPS信息等EXIF数据
提取完成后，所有信息将汇总至CSV文件，日志区可查看处理结果及文件保存路径

# 批量复制/移动文件
切换至「批量复制/移动文件」标签页，默认选择「copy（复制）」操作类型
先选择**源目录**（需要复制/移动的文件所在目录），再选择**目标目录**（文件要保存的目录）
按需切换操作类型：复制（源文件保留）、移动（源文件移除）
可填写排除扩展名（逗号分隔，如`zip,log`），过滤无需处理的文件
点击「开始执行」，文件将批量处理至目标目录，重名文件自动添加时间戳避免覆盖，日志区查看处理结果

⚠️ 注意事项<br>
处理大批量 / 大体积文件时，请勿关闭程序，等待进度条加载完成后再操作
所有功能均生成新文件，原文件不修改、不覆盖，可放心处理重要文件
建议处理核心文件前先做备份，避免因异常操作（如权限不足）导致数据问题
Windows 系统若出现杀毒软件拦截，将 exe 文件加入信任列表即可正常运行
图片加水印 / 格式转换后，文件会添加专属前缀 / 后缀，便于区分原文件<br>

## 🛠️ | 技术模块         | 所用库 / 工具                | 功能作用                                                                 |
|------------------|-----------------------------|--------------------------------------------------------------------------|
| 图形化界面       | PyQt5                       | 多标签页 GUI 开发、交互逻辑实现（新增时间选择、文件复制/移动交互）|
| 图片处理         | Pillow (PIL)                | 图片格式转换、水印添加、通道处理、EXIF 信息提取、图片分辨率/元数据解析    |
| 文件路径处理     | pathlib                     | 跨平台文件路径解析、操作（统一文件/目录有效性校验）|
| 文件压缩         | zipfile                     | ZIP 包创建、文件压缩归档（支持排除指定扩展名）|
| 正则处理         | re                          | 文件名正则匹配与替换（增加正则语法校验，避免执行异常）|
| 多线程           | PyQt5.QThread               | 后台任务执行，防止界面卡顿（扩展支持时间修改、EXIF提取、复制/移动任务）|
| 进度展示         | PyQt5.QProgressBar          | 处理进度实时可视化（统一按有效文件数计算进度，精准到1%）|
| 时间处理         | datetime / time             | 文件时间戳修改、重名文件时间戳生成、拍摄时间解析                         |
| 数据导出         | csv                         | EXIF 信息结构化导出为 CSV 文件，支持中文编码                             |
| 文件操作         | shutil / os                 | 批量文件复制/移动、文件时间修改（兼容 Windows/Linux 跨平台）|
| 异常处理         | 原生异常捕获                | 权限校验、文件占用检测、正则语法错误捕获，保证任务稳定性                 |

### 新增技术点说明
1. **datetime / time**：新增用于文件时间批量修改、重名文件自动加时间戳（避免覆盖）、EXIF 拍摄时间解析，解决时间相关操作的跨平台兼容性问题；
2. **csv**：专门用于 EXIF 信息的结构化导出，支持 UTF-8 编码确保中文正常显示；
3. **shutil / os**：补充文件复制/移动的底层操作，结合 pathlib 实现跨平台的文件操作逻辑；
4. **增强型异常处理**：在所有核心函数中增加针对性异常捕获（权限、文件占用、语法错误等），避免单个文件异常导致整个任务中断。
